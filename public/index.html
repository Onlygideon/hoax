Welcome to Hoax

This is a product management public API for e-commerce web applications, built majorly using MySQL, node.js, express, sequelize, and postman for API testing.

I built this project to showcase my ability to build well scaled APIs for frontend consumptions

This API has the following functions:-

  • Add product
  • Fetch all products
  • Fetch products based on name
  • Update product
  • Delete product
  • Create a review - rating and comment
  • Fetch all reviews for a specific product
